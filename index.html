Front End Web Developer

Get started as a front-end web developer using HTML, CSS, JavaScript, jQuery, and React!

What does a front-end web developer do?

A front-end web developer codes the front-end or graphical interface of a website. They typically take designs/mock-ups and create responsive, functional and beautiful interfaces using web technologies, including HTML, CSS, Javascript and JQuery.

The goal of this Front-End Web Developer programme is to provide the learner with the skills to build and develop high quality interactive websites delivering exceptional user experience that will make you stand out from the crowd and help secure employment as a junior front end web developer in industry.
