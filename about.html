About the course

Course Outline

The programme begins with an introduction to HTML, which provides the structure to your webpage. Once the foundation has been laid, CSS is introduced to allow you create rules that specify how the HTML content is presented or displayed on screen. Javascript adds interactivity i.e. how the page behaves, which enhances usability and user experience of webpage. The course content is outlined below:

HTML

  1. Introduction to HTML
  2. Basic Page Creation
  3. Basics
  4. Lists
  5. Links
  6. Images
  7. Tables
  8. Forms
  9. HTML Head
  10. Block and Inline Elements

CSS

  1. Introduction to CSS
  2. CSS Styling Backgrounds
  3. CSS Styling Fonts
  4. CSS Styling Text
  5. CSS Links
  6. CSS Lists
  7. CSS Tables
  8. CSS Box Model
  9. CSS Dimensions & Apply
  10. CSS Float
  11. CSS Navigation
  12. CSS Positioning
  13. CSS Align
  14. CSS Pseudo-class
  15. CSS Transformations and Transitions
  16. Designing for Mobile Devices
  17. Using Media Queries
  18. Flexbox

Javascript

  1. Getting started with Javascript Programming
  2. Working with the Document Object Model (DOM)
  3. Using Javascript Variables, Strings, and Arrays
  4. Functions, Methods, and Objects
  5. Conditions and Loops
  6. Events
  7. jQuery

Other content covered includes Search Engine Optimisation, Accessibility, and working with Content Managements Systems such as Wordpress.

About Finglas Training Centre (FTC)

Finglas Training Centre is part of the City of Dublin Education and Training Board (CDETB), providing a variety of training options for the unemployed, craft apprentices and early school leavers. It is located off the Jamestown Road in Finglas, Dublin 11.

The centre has over 200 car parking spaces including a sheltered bicycle area for students. It also has a large canteen.
